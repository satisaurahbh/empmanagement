# 👨‍💼 Employee Management System — Spring Boot

A backend Employee Management System REST API built 
with Java 21 and Spring Boot 3.x with PostgreSQL database.

---

## 🚀 About The Project

This Employee Management System allows to manage
employees, departments, salaries and leave requests through
secure REST APIs. Built to demonstrate real-world Spring Boot
development skills.

---

## 🛠️ Tech Stack

| Layer       | Technology              |
|-------------|-------------------------|
| Language    | Java 21                 |
| Framework   | Spring Boot 3.x         |
| Database    | PostgreSQL              |
| ORM         | Spring Data JPA         |
| Security    | Spring Security         |
| Build Tool  | Maven                   |
| IDE         | VS Code                 |
| API Testing | Postman                 |

---

## ✨ Features

- Add, Update, Delete Employee
- Search Employee by ID, Name, Department
- Department Management
- Salary Management
- REST API with JSON responses
- PostgreSQL Database Integration
